Pera

Pera connects Ruby on Rails with Phonegap the mobile application framework. Pera enables us to build mobile applications in Rails-like development style.

Installation

Add this line to your application's Gemfile:

gem 'kanna'

And then execute:

$ bundle

Or install it yourself as:

$ gem install kanna

Usage

Run the generator,

$ rails g kanna:install

Run guard

$ guard

Setup ios project,

$ rake kanna:ios:init

Run app on ios, (need to install ios-sim)

$ rake kanna:ios:build
